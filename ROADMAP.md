# Review Checkpoints

A complete project review should be performed:

- After each completed module.
- Before introducing a new technology.
- Before major architectural changes.
- Before production deployment.
- Whenever technical debt becomes significant.

---

# AI Rules

The AI assistant should:

- Follow the roadmap.
- Never skip unfinished phases.
- Preserve completed work.
- Recommend improvements only when they provide clear long-term value.
- Pause and recommend a project review if the architecture becomes inconsistent.
